# Claude Code Agents

A collection of specialized agents for Claude Code that extend its capabilities with domain-specific expertise and workflows.

## Agents

- **codebase-dedup-analyzer**: Performs deep analysis of codebases to identify duplicate code, unused exports, dead code branches, and unnecessary dependencies. Provides actionable refactoring recommendations with priority rankings.

- **next-dev**: Next.js 16 frontend development agent for creating components, pages, layouts, and API routes. Optimized for targeted, DRY code changes using Next.js 16 patterns including proxy.ts, Cache Components, async request APIs, and React 19.2 features.
